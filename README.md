# 🩺 PocketDr

## 🌐 Live Demo

🚀 **Try it here:** https://pocketdr.vercel.app/

---

## 📖 Overview

PocketDr is an AI-powered healthcare assistant that helps users access healthcare information through an intelligent chatbot, vaccination information, and disease outbreak monitoring.

The project was developed as part of the **Tata AI Hackathon 2026** to demonstrate how AI can improve healthcare accessibility.

## ✨ Features

- 🤖 AI-powered healthcare chatbot
- 💬 Natural language conversations using Gemini AI
- 🗺 Disease outbreak map
- 💉 Vaccination information
- 🔐 Secure user authentication
- ☁ Supabase backend integration
- 📱 Responsive user interface

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Nehal180/PocketDr.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
